## 课程心得
- 沈老师讲的英文都很简单，重要知识点都会用中文再讲一次
- 课上会有随堂小测，交了就满分
- 大作业在期末考试后再提交
- 期末考试会有20道判断题，根据1字班和2字班的经验，这20道判断题的答案都是"X"
- 考题和往年题相似
- 林今老师的自控课没有期末考试，对应分数用大作业来代替，感觉大作业的给分一般会比笔试好?
